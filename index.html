Hypertext Markup Language

Die Hypertext Markup Language, abgekürzt HTML, ist eine textbasierte Auszeichnungssprache zur Strukturierung digitaler Dokumente wie Texte mit Hyperlinks, Bildern und anderen Inhalten. HTML-Dokumente sind die Grundlage des World Wide Web und werden von Webbrowsern dargestellt. Wikipedia

Formattyp: Dokumentenformat

Aktuelle Version: 5.2 (Stand: 14. Dezember 2017)

Entwickelt von: World Wide Web Consortium (W3C)

Standard(s): ISO/IEC 15445 W3C HTML 5 W3C HTML 4.01 W3C HTML 3.2

MIME-Type: text/html

Art: Auszeichnungssprache
